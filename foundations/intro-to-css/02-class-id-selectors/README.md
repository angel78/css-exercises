# Class and ID Selectors

Luego, deberá agregar reglas en el archivo CSS proporcionado utilizando la sintaxis de selector correcta. Examine detenidamente la imagen de resultado e intente recordar qué elementos tienen un estilo similar (clases), cuáles son completamente únicos (ID) y cuáles presentan ligeras variaciones (varias clases).

* **The second element**: blue text and a font size of 36px
